以前大麻を使用していたときに、大麻精神病（cannabis psychosis）のような中毒症状をたびたび経験したため、そのときみたものを手記として公開することにしました。

幻覚剤を題材にしたオルダス・ハクスリーの『知覚の扉』がきっかけとなりました。
書き上げたものについて、エッセイのような着地点や物語のような起承転結はなく、むしろ転しかありません。
思い浮かんだ原文は泥酔しながら入力したもので、生々しさを失わないようにしながら、かつ、伝わるように書くことが困難だった場面では、わかりやすさよりも空気感を優先しました。最後のほうでは修正を一切入れずに書いている部分もあります。

解離や幻覚、妄想、自傷、あるいは病跡学（pathography）の文脈で扱われそうな描写が含まれます。
大麻精神病のような症状と前述しましたが、適切な表現が見つからないという部分が多く、専門家の診断によるものではないこと、実際の患者のかたを揶揄する意図ではないことをここに注記します。
